📋 Como Escolher

  • Tamanho divisão: 12W para 8m², 24W para 15m², 40W para 25m²
  • Temperatura cor: 2700K quente (quarto), 4000K neutro (sala), 6500K frio (cozinha)
  • LED integrado mais eficiente e sem manutenção
  • Regulável útil para criar ambientes
  • Diâmetro: 30cm pequeno, 40cm médio, 50cm+ grande
  • Altura teto: flush mount para baixos, pendente para altos

❓ Perguntas Frequentes

Quantos watts preciso?

Regra geral: 1.5W LED por m². Quarto 12m² precisa ~18W. Sala 20m² precisa ~30W. Cozinha pode precisar mais (luz trabalho).

LED integrado ou casquilho?

LED integrado dura 15-25 anos, mais eficiente, sem manutenção. Casquilho permite trocar lâmpada mas menos eficiente.

Precisa de eletricista?

Sim. Instalação requer ligação ao circuito iluminação e fixação segura no teto. Não é DIY seguro.
