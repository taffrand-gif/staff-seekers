Melhores Cabos Elétricos para Instalação 2026

O cabo elétrico é a base de toda a instalação. Escolher a secção e tipo corretos é fundamental para segurança e desempenho.

#1 — Cabo H07V-U 1.5mm² (iluminação)

Cabo rígido unifilar para circuitos de iluminação. Fácil de passar em tubo. Suporta até 16A.

Preço aproximado: ~0.30€/m

#2 — Cabo H07V-U 2.5mm² (tomadas)

Para circuitos de tomadas gerais. Suporta até 20A. O mais usado em instalações residenciais.

Preço aproximado: ~0.50€/m

🛒 Ver na Amazon.es →

#3 — Cabo H07V-U 4mm² (forno/AC)

Para equipamentos de alta potência: forno, placa, ar condicionado. Suporta até 25A.

Preço aproximado: ~0.80€/m

🛒 Ver na Amazon.es →

#4 — Cabo H07RN-F 3x2.5mm² (exterior)

Cabo flexível com bainha de borracha para exterior. Resistente a UV, água e abrasão.

Preço aproximado: ~2€/m

🛒 Ver na Amazon.es →

📋 Como Escolher

  • 1.5mm² para iluminação (disjuntor 10A)
  • 2.5mm² para tomadas gerais (disjuntor 16A)
  • 4mm² para forno, placa, AC (disjuntor 20-25A)
  • 6mm² para alimentação quadro secundário
  • H07RN-F obrigatório para exterior

🏆 A Nossa Recomendação

Compre em rolo de 100m para poupar. H07V-U para interior, H07RN-F para exterior. Nunca use cabo inferior à secção regulamentar.
