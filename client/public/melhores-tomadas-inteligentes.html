📋 Como Escolher

  • Protocolo: Wi-Fi (direto) ou Zigbee (precisa hub, mais eficiente)
  • Medição consumo útil para monitorizar gastos
  • Capacidade: 16A (3680W) suficiente para maioria aparelhos
  • Tamanho: compactas não bloqueiam tomada adjacente
  • Compatibilidade: verificar Alexa, Google, HomeKit
  • Automações locais funcionam sem internet

🏆 A Nossa Recomendação

Para Wi-Fi: TP-Link Tapo P110 (melhor preço). Para avançados: Shelly Plug S (API aberta). Para HomeKit: Meross MSS310.

❓ Perguntas Frequentes

Tomada inteligente gasta muita eletricidade?

Não. Consumo standby 0.5-2W. Menos de 0.05€/mês. Compensa pela poupança em aparelhos esquecidos ligados.

Funciona sem internet?

Depende. Controlo local (botão físico) sempre funciona. Automações locais (Shelly) funcionam. Controlo remoto precisa internet.

É seguro?

Sim, se marca certificada. Proteção sobrecarga e sobrecalor. Não usar em aparelhos >3000W (aquecedores potentes). Verificar certificação CE.
