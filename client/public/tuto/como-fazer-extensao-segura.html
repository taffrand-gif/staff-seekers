🔌 Como Fazer uma Extensão Elétrica Segura

📅 Última atualização: 25 de fevereiro de 2026

Extensões mal feitas causam incêndios. Este guia mostra como fazer com segurança.

🛠️ Material Necessário

  • Cabo elétrico (secção adequada)
  • Ficha macho (com terra)
  • Tomada fêmea ou régua (com terra)
  • Alicate de corte
  • Chave de fendas
  • X-ato ou descarnador de fios

📏 Escolha do Cabo

Secção (Espessura)

  • 1,5mm²: Máximo 10A (2300W) — iluminação, carregadores
  • 2,5mm²: Máximo 16A (3680W) — uso geral (recomendado)
  • 4mm²: Máximo 25A (5750W) — aquecedores, ferramentas

Tipo de Cabo

  • H05VV-F: Interior, flexível (mais comum)
  • H07RN-F: Exterior, borracha (resistente água)
  • 3G: 3 condutores com terra (obrigatório)

Comprimento

  • Até 10m: Sem problemas
  • 10-25m: Use 2,5mm² mínimo
  • >25m: Use 4mm² (queda de tensão)

🔧 Montagem

1. Prepare o Cabo

  1. Corte no comprimento desejado
  2. Descarne 8cm de cada ponta
  3. Descarne 1cm de cada fio
  4. Torça as pontas dos fios

2. Ligue a Ficha

  1. Abra a ficha
  2. Passe o cabo pelo corpo
  3. Conecte os fios:
    • Castanho/Preto: Terminal direito (fase)
    • Azul: Terminal esquerdo (neutro)
    • Verde-Amarelo: Terminal central (terra)
  4. Aperte bem os parafusos
  5. Fixe o cabo com a braçadeira
  6. Feche a ficha

3. Ligue a Tomada

Mesmo processo na outra ponta. Atenção: mantenha a mesma ordem de cores.

4. Teste

Use testador de tomadas ou multímetro para confirmar ligação correta antes de usar.

⚠️ Regras de Segurança

Nunca Faça

  • Ligar extensões em série: Sobrecarga e sobreaquecimento
  • Passar sob tapetes: Aquece e ninguém vê
  • Usar cabo danificado: Risco de choque
  • Sobrecarregar: Respeite a potência máxima
  • Deixar enrolada em uso: Aquece por indução
  • Usar em exterior sem proteção IP: Água + eletricidade = perigo

Sempre Faça

  • Use cabo com terra (3 fios)
  • Aperte bem todos os parafusos
  • Desenrole completamente antes de usar
  • Verifique aquecimento após 30min de uso
  • Substitua se danificado

🔌 Réguas vs Extensões

Régua Simples

  • Múltiplas tomadas
  • Limite: 3680W total (16A)
  • Sem proteção

Régua com Proteção

  • Interruptor on/off
  • Proteção contra sobretensão
  • Fusível térmico
  • Recomendado para eletrónica

Extensão Enrolável

  • Prática para guardar
  • Desenrolar sempre antes de usar
  • Enrolada: máximo 1000W
  • Desenrolada: potência total

📊 Cálculo de Carga

Exemplo

Extensão 16A (3680W máximo):

  • Computador: 300W
  • Monitor: 50W
  • Impressora: 100W
  • Candeeiro: 60W
  • Total: 510W ✅ Seguro

Perigo

  • Aquecedor: 2000W
  • Chaleira: 2000W
  • Total: 4000W ❌ Sobrecarga!

🏗️ Extensões Profissionais

Para obras e ferramentas:

  • Cabo 2,5mm² ou 4mm²
  • Tomadas IP44 (proteção água)
  • Disjuntor integrado
  • Diferencial 30mA

⚠️ Quando Chamar um Profissional

Contacte a nossa equipa se:

  • Precisa de extensão permanente (melhor instalar tomada)
  • Quer extensão para exterior
  • Precisa de extensão trifásica
  • Quer extensão com proteção diferencial
  • Usa ferramentas de alta potência
