📋 Como Escolher

  • Protocolo: Zigbee (precisa hub, baixo consumo) ou Wi-Fi (direto, mais bateria)
  • Alcance: 5-7m suficiente para divisões normais
  • Ângulo deteção: 120° mínimo, 170° ideal
  • Sensor luminosidade útil para acionar só no escuro
  • Bateria: mínimo 1 ano autonomia
  • Compatibilidade: verificar com sistema domótica existente

🏆 A Nossa Recomendação

Para Philips Hue: Hue Motion Sensor. Para HomeKit: Aqara P1. Para Wi-Fi direto: Shelly Motion 2. Para melhor preço: Sonoff SNZB-03.

❓ Perguntas Frequentes

Sensor de movimento gasta muita bateria?

Não. Modelos Zigbee duram 2-5 anos com pilhas CR2450. Wi-Fi dura menos (6-12 meses) mas alguns são recarregáveis USB.

Funciona para segurança?

Sim, mas sensores dedicados alarme são mais sensíveis. Sensores domótica servem para automação e segurança básica.

Deteta animais de estimação?

Depende. Animais >10kg podem acionar. Sensores alarme profissionais têm imunidade animais até 25kg.

Precisa de automação de iluminação?
