📋 Como Escolher

  • Potência: 20W pequenas áreas, 50W médias, 100W+ grandes áreas
  • Lúmens: mínimo 80 lúmens/W (eficiência)
  • IP rating: IP65 mínimo para exterior
  • Temperatura cor: 4000K neutro, 6500K frio (mais brilhante)
  • Sensor movimento útil para segurança e poupança
  • Corpo alumínio dissipa calor melhor que plástico

❓ Perguntas Frequentes

Quantos watts preciso?

20-30W para entrada/garagem pequena. 50W para quintal médio. 100W+ para campos desportivos ou grandes áreas.

Projetor LED gasta muito?

Não. 50W ligado 8h/dia custa ~1€/mês. 80% mais económico que halogéneo equivalente (500W).

Precisa de eletricista para instalar?

Sim, recomendado. Ligação 230V requer conhecimentos. Instalação incorreta é perigosa. Fixação segura essencial.
