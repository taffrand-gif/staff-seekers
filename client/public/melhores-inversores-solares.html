📋 Como Escolher

  • Potência: dimensionar 80-90% potência pico painéis
  • Eficiência: mínimo 96%, ideal 98%+
  • Tipo: string (standard), híbrido (com bateria), microinversor (por painel)
  • Monitorização: Wi-Fi integrado essencial
  • Garantia: mínimo 10 anos
  • Certificação: CE obrigatória, IEC 62109 ideal

❓ Perguntas Frequentes

Inversor híbrido vale a pena?

Sim se planeia adicionar baterias. Permite autoconsumo noturno. Mais caro inicialmente mas parranjado para futuro.

Microinversor vs string?

Microinversor melhor para telhados com sombra ou orientações diferentes. String mais económico para instalações simples sem sombra.

Precisa de eletricista certificado?

Sim, obrigatório. Instalação solar requer certificação DGEG. Ligação à rede precisa aprovação distribuidora. Não é DIY.
