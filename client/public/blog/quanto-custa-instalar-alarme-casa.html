📊 Fatores Que Influenciam Preço

1. Complexidade

  • Trabalho simples: preço base
  • Obra civil necessária: +30-50%
  • Acesso difícil: +20-40%
  • Instalação antiga: +20-30%

2. Materiais

  • Marcas premium: +30-50%
  • Marcas standard: preço base
  • Quantidade cabos: variável

3. Urgência

  • Agendado: preço normal
  • Urgência dia: +30%
  • Urgência noite/fim semana: +50%

4. Localização

  • Bragança, Mirandela, Chaves: preço base
  • Aldeias próximas: +10-20€ deslocação
  • Zonas remotas: +30-50€ deslocação

💡 Como Poupar

1. Agendar com Antecedência

Evita taxa urgência (+30-50%)

2. Agrupar Trabalhos

Várias intervenções numa visita reduz custo/hora

3. Fornecer Materiais

Se comprar materiais, poupa margem (mas perde garantia)

4. Manutenção Preventiva

Evita reparações caras no futuro

5. Comparar Orçamentos

Pedir 2-3 orçamentos (mas não escolher só pelo mais barato!)
