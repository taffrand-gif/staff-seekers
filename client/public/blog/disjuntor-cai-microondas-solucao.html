Disjuntor Cai ao Ligar Micro-ondas: Solução 2026

📅 27 de fevereiro de 2026

Se o disjuntor cai ao ligar o micro-ondas, pode ser sobrecarga, problema no aparelho ou circuito inadequado. A nossa equipa explica.

🔍 5 Causas Principais

1. Sobrecarga no Circuito

  • Micro-ondas: Consome 1000-1500W
  • Outros aparelhos: No mesmo circuito
  • Disjuntor 16A: Aguenta ~3500W
  • Exemplo: Micro + chaleira + torradeira = sobrecarga
  • Solução: Usar tomada diferente

2. Micro-ondas com Problema

  • Fuga elétrica: Componente danificado
  • Magnetron: Defeituoso
  • Condensador: Em curto
  • Idade: Aparelho velho
  • Teste: Ligar noutro circuito

3. Disjuntor Subdimensionado

  • 10A: Insuficiente para micro-ondas
  • Pico arranque: Maior que consumo normal
  • Solução: Disjuntor 16A ou 20A
  • Circuito dedicado: Ideal para cozinha
  • Norma: Cozinha precisa circuitos adequados

4. Fiação Inadequada

  • Cabo fino: 1.5mm² insuficiente
  • Aquecimento: Fio sobrecarregado
  • Queda tensão: Dispara disjuntor
  • Solução: Cabo 2.5mm² mínimo
  • Upgrade: Instalação necessário

5. Diferencial Sensível

  • 30mA: Detecta pequenas fugas
  • Micro-ondas: Pode ter fuga normal
  • Soma: Com outros aparelhos
  • Disparo: Diferencial em vez de disjuntor
  • Solução: Verificar isolamento

🧪 Como Testar

Diagnóstico Simples

  • 1. Desligue tudo: No circuito
  • 2. Só micro-ondas: Ligue sozinho
  • 3. Se cair: Problema no aparelho ou circuito
  • 4. Teste noutro sítio: Tomada diferente
  • 5. Se não cair: Sobrecarga no circuito original

✅ Soluções

Imediatas

  • Use tomada diferente: Circuito com menos carga
  • Não use simultâneo: Outros aparelhos potentes
  • Desligue: Chaleira, torradeira antes

Definitivas

  • Circuito dedicado: Só para micro-ondas
  • Disjuntor adequado: 16A ou 20A
  • Cabo correto: 2.5mm²
  • Reparar micro: Se aparelho tem problema
  • Upgrade quadro: Se instalação antiga

❓ Perguntas Frequentes

É perigoso continuar a usar se o disjuntor cai?

Não. Se o disjuntor dispara, há um problema que pode causar incêndio ou choque elétrico. Não force o disjuntor a ficar ligado.

Posso colocar um disjuntor maior?

Não sem verificar a fiação. Um disjuntor maior sem cabo adequado é perigoso — o cabo pode sobreaquecer e causar incêndio.
