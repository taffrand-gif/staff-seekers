👁️ Sensor de Movimento para Iluminação

Ideal para corredores, garagem, exterior. A luz liga automaticamente quando deteta movimento.

Tipos de Sensor

  • PIR (infravermelho) — deteta calor corporal, o mais comum
  • Micro-ondas — mais sensível, atravessa paredes finas
  • Crepuscular + PIR — só liga à noite

🛒 Sensor PIR de teto — Amazon.es

🛒 Sensor PIR exterior IP65 — Amazon.es

Instalação Básica (Substituir Interruptor)

⚠️ Desligue o disjuntor!

  1. Retire o interruptor existente
  2. Ligue fase (castanho) → L do sensor
  3. Ligue o fio da lâmpada → saída do sensor
  4. Ligue neutro (azul) → N do sensor
  5. Ajuste sensibilidade, tempo e luminosidade

Ajustes

  • TIME — quanto tempo fica ligado (30s a 10min)
  • LUX — nível de luz para ativar (só noite ou sempre)
  • SENS — distância de deteção (3m a 12m)
